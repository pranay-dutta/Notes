## 1. String to Number

We can covert a string into numbers using the *Number() function*, *parseInt()* and *parseFlot() methods.*

```run-js
let s = "123.33"

let x = Number(s);
let y = parseInt(s);
let z = parseFloat(s);

console.log("x:",x, "y:",y, "z:",z)
```
